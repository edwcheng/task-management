# Git Root Folder Note

## IMPORTANT: Correct Git Root Folder

The correct git root folder for this project is:

```
/home/z/my-project/task-management/
```

This folder contains the `.git` directory and all project files.

## IMPORTANT: Branch Name

The main branch is **`main`**, not `master`.

```bash
# Always use main branch
git branch -m main

# When pushing, use main
git push -u origin main
```

### What NOT to use:

- `/home/z/my-project/` - This is NOT the git root
- `/home/z/my-project/frontend/` - This is NOT part of this project (separate folder)
- `master` branch - The branch is `main`, not `master`

### How to verify:

```bash
cd /home/z/my-project/task-management
pwd                  # Should show /home/z/my-project/task-management
git rev-parse --show-toplevel   # Should show /home/z/my-project/task-management
ls -la .git           # Should show the .git directory
git branch            # Should show * main
```

### Common Mistakes to Avoid:

1. **Don't** run git commands from `/home/z/my-project/` 
2. **Don't** confuse `/home/z/my-project/frontend/` with `/home/z/my-project/task-management/frontend/`
3. **Always** verify you're in the correct folder before making commits
4. **Don't** use `master` branch - always use `main`

### To reset if you accidentally use wrong folder:

```bash
# Navigate to correct folder
cd /home/z/my-project/task-management

# If git is broken, reinitialize:
rm -rf .git
git init
git branch -m main
git add -A
git commit -m "Reinitialize repo"
git remote add origin https://github.com/edwcheng/task-management.git
git push -u origin main --force
```

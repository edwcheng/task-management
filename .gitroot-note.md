## ⚠️ CRITICAL: If You Need to Reinitialize Git

**ONLY do this if the git repository is corrupted!**

```bash
# 1. Navigate to correct folder
cd /home/z/my-project/task-management

# 2. Verify you're in correct folder
pwd  # Must show /home/z/my-project/task-management

# 3. Backup current state (optional but recommended)
cp -r . /tmp/task-management-backup/

# 4. Remove old git
rm -rf .git

# 5. Initialize new git
git init

# 6. Rename branch to main
git branch -m main

# 7. Add all files
git add -A

# 8. Commit
git commit -m "Reinitialize repository"

# 9. Add remote
git remote add origin https://github.com/edwcheng/task-management.git

# 10. Fetch and verify remote state
git fetch origin

# 11. IMPORTANT: Check if remote has newer commits
git log origin/main --oneline -5

# 12. If remote has commits you need, reset to remote FIRST
#    DO NOT force push until you've merged/verified all changes!
git reset --soft origin/main  # Keeps your changes as staged
# OR
git reset --hard origin/main  # Discards your changes, uses remote

# 13. Push (use --force ONLY if you're sure)
git push -u origin main --force
```
